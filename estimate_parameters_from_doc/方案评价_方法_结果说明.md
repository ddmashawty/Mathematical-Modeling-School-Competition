# 参数估计方案评价、实现方法与结果说明

## 一、方案可行性评价

本次实现完全从《参数计算办法.docx》和仓库已有数据出发，不调用仓库中原有模型脚本。整体方案可行，原因如下：

1. 数据基础完整。仓库中已有国内调价数据、WTI 日频油价、Brent 日频油价、Basket 日频油价和人民币兑美元汇率数据，可以支撑参数估计。
2. 权重网格搜索是合理的。由于 `w1, w2, w3` 进入综合油价 `Ot`，并且后续存在 40、80、130 美元分段处理，直接把权重作为线性回归变量不合适。先固定权重，再估计 `alpha, beta, pi0` 是稳妥做法。
3. 剔除特殊调控和累计未调价影响是必要的。基础参数应尽量由正常机制样本估计，因此只使用当前非特殊、上一期非特殊、上一期正常调价、当前正常调价的干净样本。
4. 特殊调控系数 `mu_i` 适合事后估计。特殊调控样本不参与基础参数估计，参数确定后再用特殊样本反推 `mu_i`，符合文档要求。
5. 汇率不需要分段近似。汇率是已知观测变量，可以直接并入线性回归自变量中，模型仍保持线性。这比把汇率分段近似为常数更少引入误差。

主要风险：

1. WTI、Brent、Basket 之间高度相关，可能出现多组权重误差接近的情况。
2. 特殊调控样本数量较少，`mu_i` 的稳定性判断只能作为经验结论。
3. 2026 年干净样本很少，因此年度分段参数仅作为诊断，不作为全局参数来源。
4. 如果实际政策使用的调价窗口、公告日/生效日口径、汇率口径与当前数据处理不同，误差会受到影响。

## 二、采用的实现方法

实现脚本为：

`结果/代码/estimate_parameters_from_doc.py`

核心流程：

1. 读取国内成品油调价数据、三种国际油价日频数据和汇率数据。
2. 以公告日为锚点，对 WTI、Brent、Basket 分别计算最近 10 个可得交易日均价。
3. 合并公告日前最近可得汇率。
4. 分别对汽油和柴油构造干净样本。
5. 对 `w1, w2, w3` 做步长为 `0.02` 的网格搜索，并满足 `w1 + w2 + w3 = 1`。
6. 对每组权重计算：

```text
Ot = w1 * WTI_t + w2 * Brent_t + w3 * Basket_t
O_filtered = max(40, Ot)
```

7. 固定权重后，用线性回归估计：

```text
P_actual = alpha * X_t + pi0 * H_t + beta + error
X_t = exchange_rate_t * barrel_per_ton * O_filtered
H_t = h(O_filtered)
```

其中利润因子：

```text
h(O) = 1,                  O <= 80
h(O) = (130 - O) / 50,     80 < O <= 130
h(O) = 0,                  O > 130
```

8. 用全部非特殊调控样本计算完整静态模型预测误差。
9. 分别输出按价格损失和价格波动损失选择的最优参数。
10. 用最优基础参数计算特殊调控样本的 `P_rule` 和逐次 `mu_i`。
11. 输出汽油和柴油的网格搜索结果、最优参数、分段诊断、预测结果、特殊调控系数和综合报告。

## 三、主要估计结果

### 汽油

- 最优权重：`w1 = 0.08`, `w2 = 0.92`, `w3 = 0.00`
- `alpha_gasoline = 1.219253`
- `beta_gasoline = 3946.2070`
- `pi0_gasoline = 665.5908`
- 完整静态模型 MAE：`108.2257`
- 特殊调控 `mu_i` 有效样本数：`11`
- 特殊调控 `mu_i` 的 CV：`1.1972`
- 结论：汽油特殊调控 `mu_i` 不建议近似为常数。

### 柴油

- 最优权重：`w1 = 0.08`, `w2 = 0.92`, `w3 = 0.00`
- `alpha_diesel = 1.181488`
- `beta_diesel = 3079.4006`
- `pi0_diesel = 653.7068`
- 完整静态模型 MAE：`97.9008`
- 特殊调控 `mu_i` 有效样本数：`11`
- 特殊调控 `mu_i` 的 CV：`1.1815`
- 结论：柴油特殊调控 `mu_i` 不建议近似为常数。

说明：价格损失和价格波动损失选出的最优参数一致。原因是本实现中价格波动误差定义为：

```text
(P_pred - P_prev_actual) - (P_actual - P_prev_actual)
```

化简后等于：

```text
P_pred - P_actual
```

因此价格损失和价格波动损失在 MAE/RMSE 排序上相同。
